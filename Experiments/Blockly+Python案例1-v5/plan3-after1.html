创建列表

序列是Python中最基本的数据结构。序列中的每个元素都分配一个数字作为它的位置，或者叫做索引，第一个索引是0，第二个索引是1，依此类推

Python有6个序列的内置类型，但最常见的是列表和元组

基本语法：

变量 = list() 或 变量 = [] 或 变量 = [value1, value2, value3, ...]

其中value必须可以为可变对象

例子：


a = list()
b = []
c = [1, 2]
d = [1, '2']
e = [1, '2', [3, 4]]
f = [1] * 3
g = [[0] * 3] * 3
h = [1,2,3,4,5][1:3]
i = '1,2,3,4'.split(',')
print(a, 'a is list:', type(a) == list)
print(b, 'b is list:', type(b) == list)
print(c, 'c is list:', type(c) == list)
print(d, 'd is list:', type(d) == list)
print(e, 'e is list:', type(e) == list)
print(f, 'f is list:', type(f) == list)
print(g, 'g is list:', type(g) == list)
print(h, 'h is list:', type(h) == list)
print(i, 'i is list:', type(i) == list)

其中

  1. a : 创建一个空列表
  2. b : 创建一个空列表
  3. c : 创建一个有两个数字[1,2]的列表
  4. d : 创建一个有一个数字1和一个字符2的列表
  5. e : 创建一个有一个数字1、一个字符2、一个[3.4]的列表
  6. f : 创建一个有3个重复数字1的列表
  7. g : 创建一个3*3的全0列表
  8. h : 从列表中切片
